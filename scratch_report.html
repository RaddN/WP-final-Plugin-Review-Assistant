**Dynamic AJAX Product Filters for WooCommerce v1.6.2 Review Summary** =========================================================== ### Security & Code Quality Rating **Highly Urgent - Needs Immediate Security Fixes** The plugin has multiple high-severity security issues that need to be addressed promptly, including lack of nonce verification in several AJAX handlers, which can lead to CSRF vulnerabilities. ### Top Architectural & Security Risks * [HIGH] Lack of nonce verification in five AJAX handlers (lines 2391, 2392, 3420, and 4008) in `dynamic-ajax-product-filters-for-woocommerce.php`, making it vulnerable to CSRF attacks. * **No Critical severity issues found** ### Actionable Fix Plan To ensure the plugin's security and stability, we recommend the following fixes: 1. Verify nonces using `check_ajax_referer('my-ajax-nonce')` or `wp_verify_nonce()` in all AJAX handlers (lines 2391, 2392, 3420, and 4008) in `dynamic-ajax-product-filters-for-woocommerce.php`. 2. Cast variables before merging arrays to prevent PHP errors: `array_merge( (array) $styles, ... )` or guard the variables. Please prioritize these fixes to ensure the plugin's security and code quality.

Installation:
  1. python main.py (or double-click run.bat)
  2. pip install -r requirements.txt (auto-install via run.bat)
  3. Select plugin and LocalWP site
  4. Run review
  5. Export reports

Building Standalone Executable:
  .\build.ps1
  # Creates: dist\WP-Plugin-Review-Assistant.exe
